# GeoGuessr Game Analyzer

A powerful, interactive dashboard designed to visualize and analyze your GeoGuessr match history. Gain deep insights into your performance across different countries.

![GeoGuessr Analyzer Screenshot Placeholder](https://github.com/poon919/geoguessr-game-analyzer-page/blob/master/app-screenshot.png)

## ✨ Features

- **🗺️ Interactive Map View**: Visualize your guesses vs. actual locations with interactive map.
- **📊 Advanced Performance Analytics**: Identify your strongest and weakest countries, discover "bait" locations, and pinpoint performance blind spots.
- **🖼️ Round Detail**: Deep dive into specific rounds with embedded Google Street View.

## 📦 How to Use

1. **Export your data**: Use the provided Tampermonkey script (instructions in the app) to export your GeoGuessr match history.
2. **Upload**: Drag and drop your exported JSON file into the analyzer.
3. **Analyze**: Use the sidebar to filter games, explore the map, and view performance charts.
